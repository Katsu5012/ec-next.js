# スキルルール読み込み必須（スキップ厳禁）

## タスク開始前に必ず該当スキルのルールファイルを全て読み込むこと。読み込まずに作業を開始してはならない。

### Reactコンポーネントに関わる作業（新規作成・修正・レビュー・リファクタ）

以下のルールファイルを**全て**読み込んでから着手すること：

1. `.agents/skills/vercel-react-best-practices/rules/*.md`（約50ファイル）
2. `.agents/skills/vercel-composition-patterns/rules/*.md`（8ファイル）

### UIレビュー・アクセシビリティに関わる作業

以下を読み込んでから着手すること：

3. `.claude/skills/web-design-guidelines/SKILL.md`

## 手順

1. タスクを受けたら、**コードを書く前に**上記ルールファイルを読み込む
2. ルールの内容を理解した上で、コードの作成・修正に着手する
3. 作業中もルールに違反していないか常に確認する
4. ルールを読まずにコードを書き始めた場合、それは手順違反である
